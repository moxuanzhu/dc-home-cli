# DC Home

> 数字视觉的开发主页，主要包括说明、示例、文档等。

> 如果主页出现问题请联系 [Caven Chen](https://github.com/cavencj)

> 该主页代码任何个人和机构可以修改、重构，无需经过我方授权。在进行二次开过程中出现任何问题，我方不担任何责任。

## 网站地址

> http://dc.dvgis.cn

## 安装

```node
yarn install
```

## 启动

```node
 yarn run serve
```

## 发布

```node
yarn run build
```
